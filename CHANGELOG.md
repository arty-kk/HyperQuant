# Changelog

## 1.0.0 — 2026-03-29

Initial public release.

### Included
- renamed and unified the public vocabulary around `codebook`, `vector`, `context`, and `resident`;
- removed old internal branding from the public surface;
- published a clean `hyperquant` package with OSS-ready repository hygiene;
- added Apache-2.0 licensing with a project `NOTICE`;
- shipped route benchmarking, resident-memory planning, resident-store verification, and a checked-in evidence pack;
- kept the vector route residual-rescue side channel and the resident-store payload integrity checks.

### Notes
- this release is positioned as a serious pilot-grade OSS build, not a universal performance claim;
- the strongest product path is resident-memory reduction for bounded hot-window serving.
