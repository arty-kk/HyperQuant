# Code of Conduct

This project is committed to a respectful and constructive open-source community.

## Expected behavior

- assume good intent,
- criticize ideas, not people,
- keep technical disagreement specific and evidence-based,
- avoid harassment, abuse, or exclusionary behavior.

## Unacceptable behavior

- personal attacks,
- harassment or intimidation,
- deliberate disruption,
- publishing private information without permission.

## Enforcement

Repository maintainers may remove comments, issues, pull requests, or access that violate these rules.
